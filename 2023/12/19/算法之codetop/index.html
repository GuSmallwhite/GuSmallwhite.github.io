206. 反转链表 - 力扣（LeetCode）

根据评论区的要求，迭代和递归都要会

迭代：

public ListNode reverseList(ListNode head) {
    ListNode pre=null;
    ListNode cur=head;
    while(cur!=null)
    {
        ListNode after=cur.next;
        cur.next=pre;
        pre=cur;
        cur=after;
    }
    return pre;
}

递归：

public ListNode reverseList(ListNode head) {
    if(head==null)
    {
        return null;
    }
    if(head.next==null)
    {
        return head;
    }
    ListNode p=reverseList(head.next);
    head.next.next=head;
    head.next=null;
    return p;
}

146. LRU 缓存 - 力扣（LeetCode）

这题才刷过一遍，哎，南蚌，我就记得是借助LinkedHashMap，他本身是有序的

注意点：LinkedHashMap是按照value大小排序的，但是这不意味着他能自动实现lru！你得手动实现刚被用过的放到队首

这边我们把队首放在map的尾部

然后注意：要使用迭代器进行remove！

class LRUCache {
    LinkedHashMap<Integer,Integer>mp;
    int cap;
    public LRUCache(int capacity) {
        mp=new LinkedHashMap<>();
        cap=capacity;
    }
    
    public int get(int key) {
        if(!mp.containsKey(key))
        {
            return -1;
        }
        makeRecent(key);
        return mp.get(key);
    }
    
    public void put(int key, int value) {
        if(mp.containsKey(key))
        {
            mp.put(key, value);
            makeRecent(key);
            return;
        }
        if(mp.size()>=this.cap)
        {
            int old=mp.keySet().iterator().next();
            mp.remove(old);
        }
        mp.put(key, value);
    }

    public void makeRecent(int key)//就是把他拿出来再放到尾部
    {
        int val=mp.get(key);
        mp.remove(key);
        mp.put(key, val);
    }
}

手撕快速排序

有一说一思路很清楚，但是之前没写过

class Solution {
    //快排的核心思想，就是找一个pivot，然后一轮排序就是把比他大的全放到他后面，小的全放到他前面，然后递归排序他的左右
    public int[] sortArray(int[] nums) {
        quickSort(nums, 0, nums.length-1);
        return nums;
    }
    public void quickSort(int[]nums,int start,int end)
    {
        if(start>=end)
            return;
        int i=start,j=end;
        int base=nums[start];//pivot就设置成第一个就行了
        while (i<=j) {
            //无时无刻不要判断i和j的关系
            while(i<=j&&nums[i]<base)
            {
                i++;
            }
            while(i<=j&&nums[j]>base)
            {
                j--;
            }
            if(i<=j)//交换
            {
                int t=nums[i];
                nums[i]=nums[j];
                nums[j]=t;
                i++;
                j--;
            }
        }
        //递归左右
        quickSort(nums, start, j);
        quickSort(nums, i, end);
    }
}

21. 合并两个有序链表 - 力扣（LeetCode）

public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
    ListNode res=new ListNode(0);
    ListNode p=res;
    while (list1!=null&&list2!=null) {
        if(list1.val<=list2.val)
        {
            p.next=list1;
            list1=list1.next;
        }else
        {
            p.next=list2;
            list2=list2.next;
        }
       p=p.next; 
    }
    
    p.next=list1==null?list2:list1;//因为本来就是连着的，所以不需要一个一个了
    return res.next;
}

53. 最大子数组和 - 力扣（LeetCode）

刚刚写过，觉得为易

//因为是连续的，所以只要和他前一个比较就行了
public int maxSubArray(int[] nums) {
    int []dp=new int[nums.length];//以i结尾的最大和
    dp[0]=nums[0];
    int res=dp[0];
    for(int i=1;i<nums.length;i++)
    {
        dp[i]=Math.max(nums[i], nums[i]+dp[i-1]);
        res=Math.max(res, dp[i]);
    }
    return res;
}

415. 字符串相加 - 力扣（LeetCode）

这个题目其实和第二题两束相加是异曲同工之妙，就是while循环同时维护carry

public String addStrings(String num1, String num2) {
    StringBuilder res=new StringBuilder("");
    int i=num1.length()-1,j=num2.length()-1;//得从尾部开始
    int carry=0;
    while(i>=0||j>=0)
    {
        int n1=i>=0?num1.charAt(i)-'0':0;
        int n2=j>=0?num2.charAt(j)-'0':0;
        int tmp=n1+n2+carry;
        carry=tmp/10;
        res.append(tmp%10);
        i--;j--;
    }
    //因为一直在尾部添加，所以得翻转了
    if(carry==1)res.append(1);//最后在看看有没有进位，因为你最后一次是不在循环内的
    return res.reverse().toString();
}

704. 二分查找 - 力扣（LeetCode）

public int search(int[] nums, int target) {
    int l=0,r=nums.length-1;
    //左闭右闭
    while(l<=r)
    {
        int mid=(l+r)/2;
        if(nums[mid]==target)
        {
            return mid;
        }else if(nums[mid]>target)
        {
            r=mid-1;
        }else
        {
            l=mid+1;
        }
    }
    return -1;
}

public int climbStairs(int n) {
    int []dp=new int[n+1];
    dp[0]=1;
    dp[1]=1;
    if(n==1)return 1;
    dp[2]=2;
    if(n==2)return 2;
    for(int i=3;i<n+1;i++)
    {
        dp[i]=dp[i-1]+dp[i-2];
    }
    return dp[n];
}

300. 最长递增子序列 - 力扣（LeetCode）

也是刚刚写过，鉴定为易

记得全部初始化为1

//重点是你要和之前所有的比
public int lengthOfLIS(int[] nums) {
    int[]dp=new int[nums.length];
    Arrays.fill(dp, 1);
    int res=dp[0];
    for(int i=1;i<nums.length;i++)
    {
        for(int j=0;j<i;j++)
        {
            if(nums[i]>nums[j])
            {
                dp[i]=Math.max(dp[i], dp[j]+1);
                res=Math.max(res, dp[i]);
            }
        }
    }
    return res;
}

3. 无重复字符的最长子串 - 力扣（LeetCode）

有一个注意点啊！就是更新left的时候，更新到哪！

比如abba这种情况，我们到abb的时候，b重复，left会变成2

//滑动窗口加上map,判断是否重复
public int lengthOfLongestSubstring(String s) {
    HashMap<Character,Integer>mp=new HashMap<>();
    int maxlen=0;
    int left=0;
    for(int i=0;i<s.length();i++)
    {
        if(mp.containsKey(s.charAt(i)))//有重复了，更新left，问题是更新到哪
        {
            left=Math.max(left, mp.get(s.charAt(i))+1);//你也别管原理了，记住~！
        }
        maxlen=Math.max(maxlen, i-left+1);
        mp.put(s.charAt(i), i);
    }
    return maxlen;
}

8. 字符串转换整数 (atoi) - 力扣（LeetCode）

这题自认为比较简单，对着要求做就行了

时时刻刻注意边界就行了

public int myAtoi(String s) {
    char[]str=s.toCharArray();
    int n=str.length;
    int idx=0;
    boolean neg=false;
    int sum=0;
    while(idx<n&&str[idx]==' ')
    {
        idx++;
    }
    if(idx==n)return 0;
    if(str[idx]=='-')
    {
        neg=true;
        idx++;
    }else if(str[idx]=='+')
    {
        neg=false;
        idx++;
    }else if(!Character.isDigit(str[idx]))
    {
        return 0;
    }

    while (idx<n&&Character.isDigit(str[idx])) {
        int dig=str[idx]-'0';
        if(sum>(Integer.MAX_VALUE-dig)/10)
        {
            return neg?Integer.MIN_VALUE:Integer.MAX_VALUE;
        }
        sum=sum*10+dig;
        idx++;
    }
    return neg?-sum:sum;
}
